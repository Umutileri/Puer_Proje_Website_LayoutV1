The primary users are university students collaborating on group projects and instructors who need to monitor group progress and administer fair peer evaluations.

The system uses a Python Flask backend for API services, a PostgreSQL database for secure data storage, and a modern HTML/CSS/Bootstrap frontend for responsiveness.

We offer flexible peer evaluation options, including named, anonymous, and context-based scoring, all calculated using a pre-tested algorithm to ensure accuracy and fairness based on documented contributions.
